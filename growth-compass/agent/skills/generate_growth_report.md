# Skill: Generate Growth Report

## Trigger
User requests a periodic summary (monthly, quarterly, pre-performance-review) or the system suggests one after significant accumulation.

## Goal
Generate a structured **growth visibility report** — the artifact that makes growth visible to the individual (and optionally, shareable with a manager or mentor).

## Input
- All growth entries in the period
- Skill map delta (what changed since last report)
- Development plan progress (if one exists)

## Process

### Step 1: Quantify the Period
- Number of growth entries captured
- New skills added to the map
- Skills that changed level
- Stretch vs. practice vs. exposure distribution

### Step 2: Identify Key Moments
Pick the 1-3 most significant growth entries — not necessarily the biggest wins. The moments that genuinely shifted something. Explain why each mattered.

### Step 3: Track Themes
What patterns show up across entries? Examples:
- "You're increasingly operating in ambiguous situations where you have to define the problem before solving it"
- "This quarter had more struggle entries than usual — and they cluster around cross-team coordination"

### Step 4: Measure vs. Plan
If a development plan exists, report progress against each priority gap.

### Step 5: Recommend Focus for Next Period
Based on trajectory and remaining gaps, what should the next period center on?

### Step 6: Generate Output

```
## Growth Report: [Period]
*Generated: [date] | Entries this period: [count]*

### Growth at a Glance
| Metric | This Period | Change |
|--------|------------|--------|
| Total entries | [count] | |
| Skills demonstrated | [count] | |
| Skills leveled up | [count] | |
| Stretch experiences | [count] | |
| New skills discovered | [count] | |

### Key Moments
**1. [Moment Title]**
[What happened, why it mattered, what it unlocked.]

**2. [Moment Title]**
[What happened, why it mattered, what it unlocked.]

### Patterns & Themes
[2-3 observations about the shape of growth this period]

### Development Plan Progress
| Priority Gap | Plan | Progress | Status |
|-------------|------|----------|--------|
| | | | 🟢 On track / 🟡 At risk / 🔴 Blocked |

### Skill Map Changes
*Skills that moved:*
- [Skill A]: [old level] → [new level] — [what drove the change]

### Focus for Next Period
**Primary theme**: [one phrase]
**Concrete priorities**:
1. [priority]
2. [priority]
3. [priority]

### Reflection Prompt
[One question that sets the direction for the next period]
```

## Edge Cases
- If there's very little data for the period, produce a "light report" and recommend more consistent capture
- If a previously strong skill shows regression, flag it honestly — this is useful data
- If the user is sharing this externally (manager, mentor), offer a "shareable summary" variant
